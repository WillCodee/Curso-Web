um middleware é uma função que fica entre a requisição do 
cliente e a resposta do servidor. 

Ele pode:
Modificar a requisição ou resposta
Executar lógica (como autenticação, logs, validação)
Encaminhar a requisição para o próximo middleware ou rota

📌 Pense nele como um “porteiro” que verifica ou prepara 
tudo antes de deixar a requisição seguir adiante.
